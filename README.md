# survey_analysis
A web application that helps send out survey campaigns on a given product.

This application is built on both ReactJS and NodeJS respectively.

front-end: `ReactJS`

back-end: `NodeJS and MongoAtlas`

tools: `Html, CSS, ReactJS, NodeJS,`
